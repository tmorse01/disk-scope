## Download DiskScope (Windows)

Do **not** use "Source code (zip)" or "Source code (tar.gz)" — those are repository source archives, not the installable app.

| File | Use |
| --- | --- |
| **DiskScope-{{VERSION}}-Setup.exe** | Recommended — run the Windows installer |
| **DiskScope-{{VERSION}}-win32-x64-portable.zip** | Portable build — unzip and run `DiskScope.exe` |
| **SHA256SUMS.txt** | Verify download integrity |

Windows SmartScreen may warn on first launch because builds are not code-signed yet.
